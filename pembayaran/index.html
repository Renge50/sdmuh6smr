Kenaikan Kelas Tahunan

Sebaiknya klik Backup Semua Data ke Excel atau backup JSON dulu sebelum menjalankan kenaikan kelas. Fitur ini akan mengubah kelas siswa secara massal.
Kelas Saat IniJumlah SiswaNaik / Pindah Ke

Mapping otomatis: 1 → 2, 1.1 → 2.1, 1.2 → 2.2, 2/2.1/2.2 → 3, 3 → 4, 4 → 5.1, 5/5.1/5.2 → 6, dan kelas 6 diluluskan lalu dikeluarkan dari Data Siswa aktif. Pembagian 2.1/2.2 atau 5.1/5.2 tetap bisa dilakukan melalui menu Data Siswa → Naik Kelas Massal.
